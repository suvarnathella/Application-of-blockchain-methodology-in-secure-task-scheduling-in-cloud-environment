# ⛓ Application of Blockchain Methodology in Secure Task Scheduling in Cloud Environment

## 📌 Simple Explanation
This project is about using blockchain technology to securely manage and schedule tasks in a cloud system.

## 🧸 Idea (Easy to Understand)
- Cloud = A big online system where tasks are stored ☁️  
- Task Scheduling = Assigning tasks properly 📋  
- Blockchain = A secure system where data cannot be changed 🔐  

👉 We combine all these to make a **safe and efficient task system**

## 🚀 What This Project Does
- Assigns tasks in a cloud environment  
- Uses blockchain to store task data securely  
- Prevents data tampering  
- Ensures transparency  

## 🛠 Technologies Used
- Python  
- Blockchain Concepts  
- Cloud Computing Basics
